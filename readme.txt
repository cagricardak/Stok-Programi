Bu proje stok takip sistemi uygulamasıdır.Kullanıcılar; kullanıcı ekleme, kullanıcı düzenleme (silme, güncelleme), stok ekleme ve stok düzenleme (silme, güncelleme) işlemlerini yapabilir.
Proje C# dili ve Windows Form kullanılarak Visual Studioda geliştirilmiştir.Veritabanı yönetimi için Sql Server kullanılmıştır.

Dosyalar
StokProgrami.sln : Projenin çözüm dosyası
StokProgrami.exe : Projenin çalıştırılabilir exe hali
stok_takip.sql : Projede kullandığım veritabanı Sql dosyası

Gereksinimler
1. .NET Runtime : Proje .NET 8.0 ile uyumlu olarak çalışmaktadır.Çalıştırılacak bilgisayarda .NET 8 Runtime kurulmalıdır.
2. Sql Server : Projedeki veritabanı işlemleri için Sql Server gereklidir.
3. Windows İşletim Sistemi

Kurulum Kılavuzu
Proje dosyalarını indirin.
stok_takip.sql dosyasını sql servera aktarın ve çalıştırın.
Bilgisayarınızda .NET Runtime kurulu değil ise kurun.
Veritabanının düzgün şekilde çalışması için projede App.config içinde bulunan Sql bağlantı bilgilerini güncelleyin.
StokProgrami.exe uygulamasını çalıştırın.
Artık uygulamayı kullanmaya hazırsınız.
